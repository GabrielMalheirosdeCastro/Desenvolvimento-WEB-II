# Atividade — Diagrama Entidade-Relacionamento do Projeto

> **Entrega vinculada ao Projeto Semestral — Fase 2 (Projeto e API REST)**

Chegou o momento de aplicar os conceitos de modelagem conceitual ao projeto do seu grupo. Nesta atividade, vocês devem criar o **Diagrama Entidade-Relacionamento (DER)** do sistema que estão desenvolvendo.

## O que entregar:

### 1. Descrição do Minimundo

- Um parágrafo descrevendo o domínio do sistema (baseado na especificação de requisitos da Fase 1)
- Identifique claramente as **entidades**, **atributos** e **relacionamentos** relevantes

### 2. Diagrama Entidade-Relacionamento (DER)

- Criado em ferramenta digital (dbdiagram.io, draw.io, Lucidchart, ERDPlus ou similar)
- Deve conter: entidades, atributos, chaves primárias, relacionamentos com cardinalidades
- Mínimo de **4 entidades** com relacionamentos significativos

### 3. Mapeamento para Modelo Relacional

- Lista das tabelas resultantes do mapeamento do DER
- Indicar chaves primárias (PK) e chaves estrangeiras (FK)
- Indicar os tipos de dados de cada coluna

### 4. Escolha e Justificativa do Banco de Dados

- Qual SGBD o grupo pretende utilizar? (PostgreSQL, MySQL, MongoDB, etc.)
- Justifique a escolha com base nas características do projeto
- Indique se pretendem usar algum ORM e qual

## Formato e Entrega

| Aspecto | Detalhe |
|---------|---------|
| **Formato** | PDF contendo o minimundo, o DER (imagem), o mapeamento relacional e a justificativa |
| **Onde entregar** | Atividade específica no AVA (a ser criada) |
| **Grupos** | Mesmos grupos do Projeto Semestral (até 4 alunos) |

## Critérios de Avaliação

| Critério | Peso |
|----------|------|
| Clareza e completude do minimundo | 15% |
| Qualidade do DER (entidades, atributos, cardinalidades) | 35% |
| Mapeamento correto para modelo relacional | 25% |
| Justificativa da escolha do banco e ORM | 15% |
| Organização e apresentação do documento | 10% |

## Dica Importante

Revisem a especificação de requisitos da Fase 1 antes de começar. Os requisitos funcionais que vocês descreveram lá são a matéria-prima para identificar entidades e relacionamentos. Se a especificação estiver incompleta, este é um bom momento para refiná-la.
